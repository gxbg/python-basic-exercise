# Python Basic Exercises

Conjunto de pequenos projetos em Python desenvolvidos para praticar lógica de programação durante estudos e atividades acadêmicas.

## Projetos incluídos

- Conversor de moedas
- Separador de centena, dezena e unidade
- Cálculo de idade em dias
- Conversão de dias para idade
- To-Do List em terminal
- Jogo de adivinhar o número secreto
- Jogo de Jokenpô (Pedra, Papel e Tesoura)

## Tecnologias utilizadas

- Python 3

## Objetivo

Praticar conceitos fundamentais de programação como:
- entrada e saída de dados
- condicionais
- loops
- manipulação de listas
- lógica de programação
